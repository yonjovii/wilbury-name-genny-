# Wilbury Name Generator

A fan-made novelty website that generates an unofficial Wilbury-style alias.

## Publish with GitHub Pages

1. Create a new public GitHub repository.
2. Upload `index.html`, `.nojekyll`, and this `README.md`.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder.
6. Save and wait a few minutes for the site to publish.

Your site will normally appear at:

`https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`

This is an unofficial fan-made project and is not affiliated with the Traveling Wilburys.
